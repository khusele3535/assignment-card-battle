#  Personas

## Персона 1: Стратегич Сүхээ (21 настай)
- **Мэргэжил:** Программ хангамжийн оюутан.
- **Зорилго:** Математик тооцоолол хийж, хамгийн хүчтэй комбог олох.
- **Хэрэгцээ:** HP болон Мана-ийн тоо маш тодорхой харагдах ёстой.

## Персона 2: Завгүй Заяа (28 настай)
- **Мэргэжил:** Дизайнер.
- **Зорилго:** Ажлынхаа завсарлагаар 5 минут тархиа амраах.
- **Хэрэгцээ:** Ойлгомжтой удирдлага, эелдэг текстүүд.
